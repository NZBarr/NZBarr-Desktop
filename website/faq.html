FAQ

Questions

Does NZBarr include NZBs or media?

No. NZBarr is a local library app. You need your own NZB files, Usenet access, and optional downloader setup.

Do I need a TMDB key?

It is strongly recommended for metadata, artwork, and better matching. Users enter their own key.

Is NZBarr commercial software?

No. NZBarr is a hobby open-source project shared under the GPL license.

Why does my OS warn me?

Early builds are not code signed yet, so macOS and Windows may show warnings on first launch.

Can I contribute?

Yes. Use GitHub issues and pull requests. Keep changes focused and avoid private data.

Where are downloads?

Downloads are published on the GitHub Releases page.
