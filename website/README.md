# NZBarr Website

This folder contains a static website for `nzbarr.com`.

## Files

- `index.html`: homepage.
- `download.html`: download links to GitHub Releases.
- `manuals.html`: links to GitHub documentation.
- `gallery.html`: screenshot gallery with lightbox.
- `support.html`: donation/support page with a PayPal placeholder.
- `faq.html`: short FAQ.
- `disclaimer.html`: public disclaimer.
- `styles.css`: site styling.
- `assets/`: images and JavaScript.

## PayPal Button

Open `support.html` and replace the placeholder block with the official PayPal button HTML.

## Uploading

Upload the contents of this `website/` folder to the web root for `nzbarr.com`.

The site is static HTML/CSS/JS and does not need PHP or a build step.
