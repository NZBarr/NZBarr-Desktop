Download NZBarr for your platform.

NZBarr is free and open source. Pick the installer for your desktop, or visit GitHub for the source code.

Installers

Choose the build for your computer.

macOS

Apple Silicon

For newer Macs with Apple chips, including M1, M2, M3, M4, and newer.

Download Apple Silicon DMG File: NZBarr-0.1.0-arm64.dmg

macOS

Intel

For older Intel-based Macs. Use this if About This Mac shows an Intel processor.

Download Intel DMG File: NZBarr-0.1.0.dmg

Windows

Windows 10 or newer

The standard Windows installer for most users and external testers.

Download Windows Installer File: NZBarr Setup 0.1.0.exe

Windows

Portable

A no-installer build for quick testing. The setup installer is the recommended Windows download.

Download Portable EXE File: NZBarr 0.1.0.exe
