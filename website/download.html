License notes

How licensing is handled.

Local validation

The app validates the license locally after talking to the license backend, so the desktop app can keep working without a complicated sign-in flow.

Separate backend

The website and license server stay separate from the app itself, which makes the desktop build cleaner and easier to distribute across platforms.
