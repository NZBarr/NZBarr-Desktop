Open-source desktop app

NZBarr

A desktop library for NZB files. Import, prepare, browse, enrich, and send NZBs to SABnzbd or NZBGet when you want to use them.

Download latest beta View source

NZBarr does not include media, Usenet access, indexer access, downloader software, TMDB artwork, or API keys.

Built For Personal NZB Libraries

NZBarr is made for users who already have their own NZB files and want a visual local library around them.

Smart Preparation

Prepare movie and TV NZBs before import, normalize filenames, and improve matching with TMDB IDs.

Metadata And Artwork

Add your own TMDB key to fetch posters, backdrops, logos, cast data, and better title matches.

Downloader Support

Send releases to SABnzbd or NZBGet after configuring your own downloader connection.

A Hobby Project, Shared Open Source

NZBarr is built by a hobby programmer, not a commercial software company. It is shared for people who find it useful, want to learn from it, or want to help improve it.
