Refresh Category Setup ▼

NZBarr uses this category when sending refresh NZBs to SABnzbd. SABnzbd routes it to a specific folder.
Path to the SABnzbd category folder for refresh jobs. Must be accessible on this machine (e.g., SMB mount).
Choose what NZBarr does with the completed refresh download after the new NZB has been uploaded.
When move is selected, NZBarr moves the completed refresh download here instead of deleting it.
One-time SABnzbd Setup:
  1. Open SABnzbd Settings → Categories
  2. Add a new category named nzbarr-refresh
  3. Set its folder to complete/nzbarr-refresh (or your preferred path)
  4. Save in SABnzbd
  5. Set the "Completed Refresh Folder Path" above to that folder on your machine
This works exactly like Sonarr/Radarr category routing — one-time setup.

Release Groups ▼

Import & Link ▼

Drag and drop NZB files anywhere to import them automatically. If a TV Show or Movie is not found at ThemovieDB, you can add it manually. Then you can link your nzb files to that TV Show/Movie. Or use the Link NZBs to Media button to manually associate NZBs with existing library entries.

Smart Preparation ▼

Prepare movie and TV NZB filenames inside NZBarr before import. This native pass resolves TMDB and IMDb IDs, normalizes naming, skips unmatched files, and can optionally import the prepared releases immediately.
